# qtawss3
Amazon S3 explorer

Features:

* object tree;
* creation of directories;
* download objects;
* upload objects.

## Requirements

* C++11 compiler support;
* Qt5;
* CMake 2.8.11.

## Build

    git clone git@github.com:RPG-18/qtawss3.git
    cd qtawss
    mkdir build
    cmake ..
    make
